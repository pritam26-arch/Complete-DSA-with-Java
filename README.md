## 📌 The Engineering Blueprint (Curriculum Tracker)

The syllabus is structured into a three-tier architecture to ensure a predictable and scalable learning path:

### 🟢 Beginner Level: Engine Initialization & Logic Building
*Focusing on core fundamentals, logic architecture, and basic data layouts.*
- [ ] **Git & GitHub Supremacy:** Trunk-based development and maintaining a clean commit history.
- [ ] **Logic Mechanics:** Breaking down complex problems using Flowcharts and Pseudocode.
- [ ] **Java Syntax Core:** Conditionals, optimized loops, switch enhancements, and clean function decomposition.
- [ ] **Complexity Analysis:** Big O, Big Omega, and Theta notations. Space and Time complexity proofs.
- [ ] **Linear Data Layouts:** Arrays and dynamic allocation arrays (ArrayLists). Under-the-hood memory management.
- [ ] **The Search Engines - Part 1:** Linear Search (best/worst case optimization) and basic Binary Search (1D Arrays).

### 🟡 Intermediate Level: Algorithmic Thinking & Paradigm Shift
*Diving into complex sorting paradigms and memory-efficient operations. The absolute sweet spot for technical interviews.*
- [ ] **Advanced Binary Search:** Searching in row-wise and column-wise sorted 2D Matrices. Handling corner cases (Infinite arrays, peak index elements).
- [ ] **The Sorting Playground:** Bubble, Selection, and Insertion Sort manual implementations.
- [ ] **Cyclic Sort Paradigm:** The ultimate cheat code for index-matching array allocation problems in O(N) time complexity.
- [ ] **Memory Efficient Strings:** String pools, deep comparison, and `StringBuilder` internal buffer management.
- [ ] **Bitwise Manipulation & Math:** Bitmasking techniques, XOR optimization hacks, Sieve of Eratosthenes, and the Euclidean algorithm.
- [ ] **Object-Oriented Design (OOPs):** Abstraction, Encapsulation, Inheritance, Polymorphism. Classes, Interfaces, and Generics.

### 🔴 Advanced Level: Low-Level Data Engineering (The Boss Fights)
*Peak algorithmic efficiency. Coding custom structures and optimizing recursive execution stacks.*
- [ ] **Recursion & Divide-and-Conquer:** Deep analysis of Execution Trees. Merge Sort, Quick Sort, Subset generation.
- [ ] **Backtracking Architecture:** N-Queens placement matrix, Knight's Tour, Rat in a Maze, Sudoku Solver.
- [ ] **Custom Linear Structures:** Low-level design of Singly, Doubly, and Circular Linked Lists.
- [ ] **Stacks & Queues Engine:** Custom implementations. Classic problems (Largest Rectangle in Histogram, Min Stack, Circular Queue).
- [ ] **Hierarchical Trees:** Binary Trees, Binary Search Trees (BST), AVL Trees (Self-balancing mechanics).
- [ ] **The Hashing Architecture:** HashMaps internal working, Custom LinkedList Chaining buckets, handling collisions for O(1) lookups.
- [ ] **Graphs & Network Topology:** Adjacency Matrix & Lists. Traversal mechanics (BFS & DFS). Shortest path algorithms (Dijkstra, Bellman-Ford, Prim’s).
- [ ] **Dynamic Programming (DP):** Overlapping subproblems layout. Memorization vs Tabulation approaches.
