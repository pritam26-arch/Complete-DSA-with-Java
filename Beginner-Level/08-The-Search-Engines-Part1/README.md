# 🔍 Module 08: Basic Search Algorithms

> "Data is useless if you can't retrieve it efficiently."

## 🎯 Objective
The entry point to real algorithmic problem-solving. This module focuses on retrieving targets from arrays and marks the transition from Brute-Force to Logarithmic optimization.

## 🧠 Core Engineering Focus
- **Linear Search:** The brute-force approach. Traversing data in $O(N)$ time. Handling edge cases in strings and 2D arrays.
- **Basic Binary Search:** The first major algorithmic optimization. Reducing search time to $O(\log N)$ by halving the search space repeatedly on sorted arrays.
- **Order-Agnostic Binary Search:** Dynamically deciding the sorting order (Ascending/Descending) before executing the search.
