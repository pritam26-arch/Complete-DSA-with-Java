# 🔍 Module 08: Basic Search Algorithms

> "Data is useless if you can't retrieve it efficiently."

## 🎯 Objective
The entry point to real algorithmic problem-solving. This module focuses on retrieving targets from arrays and marks the transition from Brute-Force to Logarithmic optimization.

## 🧠 Core Engineering Focus
- **Linear Search:** The brute-force approach in O(N) time.
- **Basic Binary Search:** The first major algorithmic optimization reducing search time to O(log N).
- **Order-Agnostic Binary Search:** Dynamically deciding the sorting order before execution.
