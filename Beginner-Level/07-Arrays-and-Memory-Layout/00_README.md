# 📊 Module 07: Linear Data Layouts & Memory

> "Mastering contiguous memory mapping is the gateway to advanced data structures."

## 🎯 Objective
Moving from single variables to structured data collections. This module goes deep into how arrays are stored in system memory and how dynamic arrays bypass static limitations.

## 🧠 Core Engineering Focus
- **Heap vs. Stack Layout:** Stack (references) vs Heap (Array objects).
- **Contiguous Allocation:** How elements are mapped back-to-back in memory.
- **Multi-Dimensional Layouts:** Traversing 2D arrays (Matrices).
- **Dynamic Resizing (ArrayList):** Amortized O(1) time complexity and internal capacity doubling.
