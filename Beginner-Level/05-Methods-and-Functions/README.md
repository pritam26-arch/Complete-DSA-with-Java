# 📦 Module 05: Modular Execution & Scope

> "Don't Repeat Yourself (DRY). Build reusable, isolated execution blocks."

## 🎯 Objective
Decomposing massive monolithic code into small, testable, and reusable methods. This module dives deep into how Java handles memory allocation during function calls.

## 🧠 Core Engineering Focus
- **Stack Frame Allocation:** Understanding how memory is pushed and popped when a method is called.
- **Pass-by-Value Architecture:** How Java handles primitive vs. object reference passing.
- **Variable Shadowing & Block Scope:** Understanding the lifetime and visibility of variables.
- **Method Overloading:** Compile-time polymorphism using different method signatures.
- **Varargs (`...`):** Handling dynamic, variable-length arguments gracefully.

*Status: Methods isolated and modularized.*
