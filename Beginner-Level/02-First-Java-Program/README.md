## 🎯 Objective
This module explores the absolute basics of Java. It's not just about printing "Hello World"; it's about understanding the entire execution pipeline from writing source code to JVM execution.

## 🧠 Core Engineering Focus
- **The Execution Pipeline:** How the JDK compiles code into Bytecode (`.class`) and the JRE/JVM executes it.
- **Static Typing & Primitives:** Memory limits of `int`, `long`, `float`, and `char`.
- **Type Conversion:** The mechanics of Implicit (Widening) and Explicit (Narrowing/Casting) type conversions.
- **Standard I/O:** Capturing user inputs dynamically using the `Scanner` class.
